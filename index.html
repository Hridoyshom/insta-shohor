How Does Javascript Work?

In our .js file we write our code that human can understand.But when we execute the code with our Browser it can not understand the human readable code.Here comes different engine for different Browser to convert the code into machine readbale form to understand the code. These engines works like compilers.There are mainly two component to execute the javascript code. one is memory component another is code component.Memory components store all the variables and functions as the key value. That's why the memory componenet is also called variable componenet. The second componenet is called code componenet. Javascript executes a single command of code at a time synchronously. That's why javascript is called single threaded programming language. The browser engine executes the code from code component it store like stack. If sometimes the stack overflows then browser show us errors.
